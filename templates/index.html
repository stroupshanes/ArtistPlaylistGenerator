Spotify Artist Playlist Generator

Login below to create individual artist playlists from each of your liked songs on Spotify.

Once you've logged in, copy & paste the access token from the url above into the form below. Your URL should look something like the below:

→ http://stroupshanes.pythonanywhere.com/#access_token=(Your Access Token Here)&token_type=Bearer&expires_in=3600 ←

Please Note: due to restrictions with the Spotify API, users need to be manually approved for use with this application. You can make a request for use by creating an issue on the Github repository.
